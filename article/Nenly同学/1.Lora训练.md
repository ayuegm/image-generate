#### 2. 第一部分：LoRA模型原理作用分析（最重要理论段，建议反复看）
UP主用最通俗的“插针”比喻讲解（比第1课更深入）：
- LoRA本质 = 在U-Net的**Cross Attention层**插入极小的“旁路矩阵”（A矩阵+B矩阵）。
- 训练时只更新这两个小矩阵（参数量只有全模型的0.1%~1%），所以显存极低（6~8GB就能训）。
- 数学简述（不吓人）：ΔW = BA（低秩分解），最终生成时W' = W + α·BA。
- 与DreamBooth、Embedding的本质区别：
  - Embedding：只改文本向量（几KB）
  - LoRA：改U-Net连接方式（几MB~200MB）
  - DreamBooth：改整个U-Net（几GB）
- LoRA优势总结：快、轻、强、可叠加、可合并、可提取（LyCORIS进阶）。

#### 3. 第二部分：LoRA训练工具配置（实操第一步）
- 推荐工具：**Kohya_ss**（GUI版，最稳定）
- 完整安装流程（视频演示+飞书文档链接）：
  1. 安装Git + Python 3.10.6
  2. 克隆仓库 `git clone https://github.com/bmaltais/kohya_ss.git`
  3. 运行 `setup.bat`（自动下载所有依赖）
  4. 启动 `kohya_gui.bat`
- 备选工具简要对比：
  - 秋葉aaaki的SD-Trainer（更美观，但参数少）
  - 朱尼酱赛博丹炉（云端友好）
  - LyCORIS（支持LoHa、LoKr等高级变种）
- 显存建议：最低6GB（用--xformers或--medvram），推荐12GB+。

#### 4. 第三部分：LoRA训练基本流程梳理 + 底模选择
**完整流程**（视频一步步演示）：
1. 准备图片文件夹（jpg/png）
2. 准备caption文件夹（.txt文件，与图同名）
3. 打开Kohya GUI → LoRA tab
4. 选择底模（.safetensors）
5. 设置文件夹路径、触发词、参数 → 开始训练

**知识点：底模版本含义与选择思路**（重中之重！）
- SD1.5系列（最推荐新手）：Anything V4.5、Realistic Vision、Counterfeit、ChilloutMix等
- SDXL（进阶）：Pony、Animagine、Juggernaut XL
- 选择原则：
  - 训角色 → 选真人/半写实底模
  - 训画风 → 选动漫底模
  - 训物品 → 选通用底模
- 绝对不要用SD2.1或早期实验版。

#### 5. 第四部分：LoRA训练参数深入剖析（参数地狱最详细讲解）
UP主把所有参数逐个拆解，并给出**新手推荐值**（视频里反复强调）：

| 参数类别          | 关键参数                  | 新手推荐值                  | 重点解释（视频原话要点） |
|-------------------|---------------------------|-----------------------------|--------------------------|
| 基础设置         | LoRA Type                | Standard LoRA              | 选Standard最稳 |
| 网络维度/Alpha   | Rank / Alpha             | 32 / 16（或128/64进阶）    | Rank越高细节越强，但过拟合风险↑ |
| 学习率           | Learning Rate            | 1e-4（0.0001）             | 最关键！太高崩，太低学不动 |
| 文本编码器LR     | Text Encoder LR          | 5e-5（学习率1/2）          | 角色训练必须开 |
| 训练步数         | Max Train Steps          | 1000~3000（按数据集大小）  | 每张图约10~20步 |
| Batch Size       | Batch Size               | 1~4                        | 显存决定 |
| Resolution       | Resolution               | 512 or 768                 | 与底模一致 |
| 保存间隔         | Save every N epochs      | 100~200                    | 方便挑最佳epoch |
| 触发词           | Trigger Word             | 自定义稀有词（如“myStyle”）| 不要用常见词 |
| 权重衰减         | Optimizer                | AdamW8bit（省显存）        | 默认即可 |

**监控训练进程知识点**：
- 看Loss曲线（目标0.1~0.2左右）
- 每200步生成测试图（视频演示如何边训边看）
- 过拟合信号：Loss突然下降但测试图开始“复制”训练集细节 → 立即停止

#### 6. 第五部分：训练数据集进阶处理技巧 + 标注清洗
这是全视频最实用、细节最密集的部分：
- **图片准备**：
  - 数量：10~50张（角色建议20~30张）
  - 要求：多样角度、表情、服装、光线，但核心特征一致
  - 裁剪：正方形或底模分辨率，去水印、文字
- **Caption技巧**（重点中的重点）：
  - 自动打标：用BLIP或WD14（视频演示）
  - **“清洗”技巧**：手动删除无关描述，保留核心特征词（如“1girl, solo, long hair, blue eyes”）
  - 添加触发词：每张caption末尾统一加 `myStyle` 或 `TOK`
  - 高级：用正则批量替换、控制词权重（如`(masterpiece:1.2)`）
- **正则化图片**（可选但推荐）：放几张“空”图（无触发词），防止过拟合背景。

#### 7. 第六部分：LoRA训练物品与画风实践（结尾实操演示）
- **物品训练示例**：训一个“猫耳发夹” → 10张图、1000步、Rank=16，效果演示（前后对比极明显）
- **画风训练示例**：训某个冷门艺术家风格 → 15张图、参数微调，生成多张风格统一图
- 训练后使用方法：
  - 文件放 `models/Lora` 文件夹
  - Prompt写法：`<lora:myStyle:0.8>` 或 `myStyle, 1girl...`
  - 权重建议0.6~1.0（太高易崩）
- 常见问题速解：过拟合、风格污染、触发词失效 → 对应调整Rank/LR/数据集
